Colors:

  1. Giving color name

    Example:

    color:red;

  2. RGB(Red, Green, Blue)

    Value Range: 0 to 255

    Syntax: rgb(0,0,0)

    Example:

    color:rgb(0,255,0)

  3. RGBA(Red, Green, Blue,Opacity)

    Opacity Range: 0.1 to 1.0

    Syntax: rgb(0,0,0,0)

    Example:

    color:rgb(0,0,255,0.5)

  4. Hex value(#RRGGBB)

    Value Range: 00 to FF

    Syntax: #000000 or #000

    Example:

    color:#00F0DA

Units:

Absolute Lengths:

  1. px(pixel):

    1px = 1/96th of 1inch

    Example:

    Text with unit of "px".

  2. pt(point):

    1pt = 1/72 of 1inch

    Example:

    Text with unit of "pt".

  3. pc(picas):

    1pc = 12 px

    Example:

    Text with unit of "pc".

Relative Lengths:

  1. em:

    By default 1em = 16px

    Relative to the font-size of the element (2em means 2 times the current font-size)

    Example:

    These paragraphs have calculated as:
    font-size is: 16px(default) * 2em = 32px.
    line-height is: 32px(2em(16*2) of its parent(p tag) font-size) * 2em = 64px.

    These div have calculated as:
    font-size is: 30px. (Because unit of this font-size is "px". So no need to look for parent.)
    line-height is: 30px(30px of its parent(div tag) font-size) * 2em = 60px. These span inside div calculated as:
    font-size is: 30px(Because font-size of this parent div is 30px) * 0.5em = 15px
    line-height is: 30px(30px of its parent(span tag) font-size) * 1em = 15px.

  2. rem:

    Relative to font-size of the root element

    Example:

    Font-size: 16px(root element's "html") font-size * 0.5rem = 8px

    Font-size: 16px(root element's "html") font-size * 1rem = 16px

    Font-size: 16px(root element's "html") font-size * 1.5rem = 24px
