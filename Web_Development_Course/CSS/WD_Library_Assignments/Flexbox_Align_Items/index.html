Points to be remember:

  • when you set the direction to a reversed row or column, justify content-start and end are also reversed.

    Example:

    flex-direction:row-reverse;
    justify-content:flex-end

  • when the flex direction is a column, justify-content changes to the vertical and align-items to the horizontal.

    Example:

    flex-direction:column; justify-content:flex-end

  • The align-content determines the spacing between lines, while align-items determines how the items as a whole are aligned within the container.
    When there is only one line, align-content has no effect.

Display Flex property - Align Items:

  • The align-items property specifies the default alignment for items inside a flexbox or grid container.

  • In a flexbox container, the flexbox items are aligned on the cross axis(y-axis), which is vertical by default.

  • Default: Stretch

Align item: "Stretch"

Items are stretched to fit the container.

Align self:

  • The align-self property specifies the alignment for the selected item vertically(y-axis) inside a flexbox container.

  • Align-self values:

    • Stretch
    • Flex-start
    • Flex-end
    • Center
    • Baseline
