Selectors:

Types of Selectors:

  • Element selector:

    It applies style for all related element in HTML webpage.

    Example

    Text color will be "indigo" for all "pre" tag content in this webpage.

  • Class selector:

    1. It apply styles only for those elements that used this "class name"

    2. A class name can be used in multiple elements.

    3. More than one classname can be used in same element.

    4. Syntax: .Class_name{}

    Example

    Text with "one" class styles applied.(ie.class1-->color)
    Text with "two" class styles applied.(ie.class1-->color & class2-->font-size)

  • ID selector:

    1. Id should be unique and used in an individual element.

    2. Id should not be repeated

    3. An element should not have 2 id's.

    4. Syntax: #ID_name{}

    Example

    Text with ID style applied.

  • Grouping selector:

    1. Grouping two or more elements, class or id styles.

    2. Syntax: Separated by ','

    Example

    Link 1 Link 2

  • Decendant selector:

    1. Apply styles for Parent-Child.

    2. Syntax: Separated by ' '(space).

    Example

    H1 inside div

    H2 inside div

  • Pseudo selector:

    1. A tag order: Love HA --> Link, visited, Hover, Active.

    2. Syntax: Tagname followed by ':' properties{}.

    Example

    Open Google

Examples using pseudo properties:

Result of p:first-child: First line of text.

Result of p:nth-child(even): Second line of text.

Result of p:nth-of-type(3): Third line of text.

Result of p:nth-child(even): Fourth line of text.

Result of p:last-child: Fifth line of text.
