  • class attribute selector:

    1. [attribute|="value"]:

      This selector is used to select elements with the specified attribute, whose value can be exactly the specified value, or the specified value followed by a hyphen (-).

      Example:

      Class name ---> para

      Class name ---> para-2

      Class name ---> para-three

      Explanation: The value has to be a whole word, like class="para", or followed by a hyphen( - ), like class="para-2".

    2. [attribute^="value"]:

      This selector is used to select elements with the specified attribute, whose value starts with the specified value.

      Example:

      Class name ---> line

      Class name ---> line-2

      Class name ---> three-line

      Explanation: The value does not has to be a whole word but starts with the word 'line'.

    3. [attribute$="value"]:

      This selector is used to select elements with the specified attribute, whose value ends with the specified value.

      Example:

      Class name ---> text

      Class name ---> text-2

      Class name ---> third-text

      Explanation: The value does not has to be a whole word but ends with the word 'text'.

      4. [attribute*="value"]:

      This selector is used to select elements whose attribute value contains a specified value.

      Example:

      Class name ---> paragraph_I

      Class name ---> paragraph_II

      Class name ---> paragraph_III

      Explanation: The value does not has to be a whole word but it should contains the word '_I'.
