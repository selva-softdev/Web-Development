Display Properties:

  1. Inline:

    • An inline element does not start on a new line.
    • An inline element only takes up as much width as necessary.
    • Width and height cannot be resized. ie,it takes only needed content space.
    • Content displayed in row.

    Inline Elements in HTML
    big small b strong i em q
    object sub sup code kbd cite time
    input output button select textarea label br
    a img map span abbr acronym script

    Example:

    Text inside "Span-1" Text inside "Span-2"

  2. Block:

    • A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
    • It will take up the full width
    • Content displayed in column.

    Block Elements in HTML
    article aside header footer main address h1 to h6
    div section p pre ol ul li
    table tfoot fieldset figure figcaption hr video
    dd dl dt video nav canvas blockquote

    Example:

    Text in "P-1"

    Text in "P-2"

  3. Inline-Block:

    • Displays an element as an inline-level block container.
    • The element itself is formatted as an inline element but height and width values can be applied.

    Example:

    Text inside "Span-1" Text inside "Span-2"

  4. Flex:

    • A parent element must have the display property set to flex.
    • Direct child elements(s) of the flexible container automatically becomes flexible items.

    Flex Direction:

    Used to format the flex items.

    Default value: "Row".
